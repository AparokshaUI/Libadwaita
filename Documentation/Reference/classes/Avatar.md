**CLASS**

# `Avatar`

Adw.Avatar

## Methods
### `init(size:text:showInitials:)`

Initialize an avatar.
- Parameters:
  - size: The avatar's size.
  - text: The avatar's text.
  - showInitials: Whether to show the initials.

### `fallbackIcon(_:)`

Set the fallback icon.
- Parameter icon: The new fallback icon.
- Returns: The avatar.

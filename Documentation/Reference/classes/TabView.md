**CLASS**

# `TabView`

Adw.TabView

## Methods
### `init()`

Initialize a tab view.

### `append(title:_:getPage:)`

Append a new tab.
- Parameters:
  - title: The tab's title.
  - widget: The tab's widget.
  - getPage: Get the tab page's identifier.
- Returns: The tab view.

### `prepend(title:_:getPage:)`

Prepend a new tab.
- Parameters:
  - title: The tab's title.
  - widget: The tab's widget.
  - getPage: Get the tab page's identifier.
- Returns: The tab view.
